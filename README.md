# 🏥 Predicting Disease Outbreak using Data Analytics

### 🚀 Overview
A professional Public Health Command Center built as a Python Full-Stack project. This application uses a **Random Forest Machine Learning model** to predict 41 different diseases based on patient symptoms. It features interactive analytics, real-time risk gauges, and geospatial hotspot detection.

### 🛠️ Key Features
* **AI-Driven Diagnoser:** Real-time symptom analysis for 41 conditions (Requirement #3).
* **Geospatial Hotspot Detection:** Interactive mapping centered on regional clusters (Requirement #5).
* **Epidemiological Trends:** Analysis of symptom importance and outbreak red-flags (Requirement #7).
* **Risk Forecasting:** 7-day predictive projections for healthcare planning (Requirement #6).

### 🧪 Tech Stack
* **Python 3.13**
* **Streamlit** (UI Framework)
* **Scikit-Learn** (ML Model)
* **Plotly & Matplotlib** (Analytics & Charts)
* **Custom CSS** (Glassmorphism UI Design)

### ⚙️ How to Run Locally
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`
